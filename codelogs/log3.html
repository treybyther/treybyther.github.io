Code Log 3: CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media query is aimed to allow for responsive design. Essentially, not all devices are the same size. So if you set the font size of your website to a specific point size, it may not come across visually as good to someone who is using a phone versus a computer. So, you use a CSS media query to allow for extra coding for devices of a certain sizes. This allows your website to have responsive design. It’s like an if statement, but for the size of the devices screen.
