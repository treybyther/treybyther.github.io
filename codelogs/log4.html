Code Log 4: CSS Color

What do these CSS properties do?

Color: This CSS property changes the color of the item that you have selected. You can change the font, background, border, margin, etc.

Background-color: The background color changes the color of the section's background, and it includes the padding and the border, but not the margins.

Border-color: This sets the border to a color, and you are able to not only set all 4 to the same, but you can choose which borders are colored, and they can all have different colors.

How does CSS accept color as a value?

Named: The name of the color you want to type. If you type red, you will get the assigned red color.

Hexadecimal: A six-digit alphanumeric code that represents the color you want to change it to.

RGB: It creates the color based on the weight of red, green, blue from 0 to 255 on all of them.
