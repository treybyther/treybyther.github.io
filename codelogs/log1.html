Code Log 1: HTML Semantic Tags

What is the difference between non-semantic and semantic tags in HTML?

The difference between non-semantic and semantic tags in HTML, are that a semantic tag is one that has meaning. Basically, it encommpasses how that specific section is going to behave. For example, if you are creating a table, and you use <table, you know that it is going to be setting up a table, and there is internal instructions on how to set that specific table up.
